# INRIX Assessment

## How to Run the Project

1. **Prerequisites**:
   - Java (JDK 8+)
   - Maven installed.

2. **Build and Run**:
   ```sh
   mvn clean install
   ```
   ```sh
   java -cp target/classes com.inrix.Main
   ```
